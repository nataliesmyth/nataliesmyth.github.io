GetMeFlightsmore_vert
GA - GetMeFlightsclose

GetMeFlights! put simply, is a flight fare alert app, much like you'd find on Kayak.com. The main focus here was to learn how to schedule background processes and having those processes act on certain conditions. Flight data was pulled from QPX Express' API. The user can then set a budget for what they'd ideally like to pay for any given flight and then the app would run a daily process to check to see whether that day's flights to location X has met the user's set budget. If met, the app would then email the user to notify them that it's time to book their flight.
